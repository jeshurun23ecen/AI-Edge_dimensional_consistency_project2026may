This project was developed as part of a project with L&T EduTech.
The objective was to classify mechanical parts as dimensionally consistent or inconsistent using machine learning.
An XGBoost classifier was trained on dimensional measurement data collected from manufactured parts.
The dataset included multiple geometric and tolerance-based parameters used for quality inspection.
The model was trained and optimized to improve classification accuracy and reduce false detections.
Performance was evaluated using metrics such as accuracy, precision, recall, and confusion matrix analysis.
This project provided practical exposure to industrial machine learning workflows and predictive analytics.
References:
1. Chen, T. & Guestrin, C. (2016). XGBoost: A Scalable Tree Boosting System.
2. Breiman, L. (2001). Random Forests.
